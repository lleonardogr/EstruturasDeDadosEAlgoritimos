- Pré-Requisitos
Python instalado: Verifique se o Python está instalado em seu sistema. Você pode verificar isso abrindo o terminal (ou prompt de comando no Windows) e digitando python --version ou python3 --version.

VSCode instalado: Se você ainda não tem o VSCode, pode baixá-lo do site oficial. Você pode seguir o tutorial do VSCode
no link: https://code.visualstudio.com/docs/python/python-tutorial

- Executando o Script no VSCode
Abra o VSCode.

Crie um novo arquivo: Vá em "Arquivo" > "Novo Arquivo". Salve este arquivo com a extensão .py, por exemplo, linked_list_calculator.py.

Copie e cole o código: Copie o script Python que forneci e cole-o no arquivo que você criou.

Execute o script: Você pode executar o script de duas maneiras no VSCode:

a. Clique com o botão direito no editor de texto e escolha "Executar arquivo Python no terminal".

b. Pressione F5 e selecione "Python" para executar o script em modo de depuração.

Visualize a saída: A saída do script será exibida no terminal integrado do VSCode.

- Executando o Script na Linha de Comando
Abra o terminal ou prompt de comando.

Navegue até a pasta onde o script está salvo: Use o comando cd para navegar até o diretório onde você salvou o arquivo .py. Por exemplo:

bash
Copy code
cd caminho/para/diretorio
Execute o script: Digite o seguinte comando no terminal:

bash
Copy code
python Calculadora.py
ou, em alguns sistemas: python3 Calculadora.py

Se você já tiver o Python dentro de suas variavéis de ambiente, apenas é necessário escrever:
python3 Calculadora.py ou python Calculadora.py

Visualize a saída: Assim como no VSCode, a saída do script será exibida no terminal.

Notas Adicionais
Se você encontrar algum erro ao tentar executar o script, verifique se o Python está corretamente instalado e se o caminho do diretório onde o arquivo .py está salvo está correto.
Certifique-se de que o script Python esteja livre de erros de sintaxe. O interpretador Python fornecerá informações sobre erros de sintaxe ou exceções, se houver algum.
O VSCode pode solicitar que você instale a extensão do Python se ainda não estiver instalada. Essa extensão facilita a escrita e execução de scripts Python no VSCode.
Seguindo esses passos, você deve ser capaz de executar o script Python tanto no VSCode quanto na linha de comando sem problemas.
